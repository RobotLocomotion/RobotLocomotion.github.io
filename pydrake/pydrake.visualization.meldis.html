pydrake.visualization.meldis¶

MeshCat LCM Display Server (MeLDiS)

A standalone program that can display Drake visualizations in MeshCat by listing for LCM messages that are broadcast by the simulation.

This can stand in for the legacy drake-visualizer application of days past.

From a Drake source build, run this as:

bazel run //tools:meldis &

From a Drake binary release (including pip releases), run this as:

python3 -m pydrake.visualization.meldis

In many cases, passing -w (i.e., --open-window) to the program will be convenient:

bazel run //tools:meldis -- -w &
class pydrake.visualization.meldis.Meldis(*, meshcat_host=None, meshcat_port=None)¶

Bases: object

MeshCat LCM Display Server (MeLDiS)

Offers a MeshCat vizualization server that listens for and draws Drake’s legacy LCM vizualization messages.

If the meshcat_host parameter is not supplied, ‘localhost’ will be used by default.

Refer to the pydrake.visualization.meldis module docs for details.

__init__(*, meshcat_host=None, meshcat_port=None)¶

Initialize self. See help(type(self)) for accurate signature.

serve_forever(*, idle_timeout=None)¶

Run indefinitely, forwarding LCM => MeshCat messages.

If provided, the optional idle_timeout must be strictly positive and this loop will sys.exit after that many seconds without any websocket connections.
