Meshcat Class Reference

Detailed Description

Provides an interface to https://github.com/rdeits/meshcat.

Each instance of this class spawns a thread which runs an http/websocket server. Users can navigate their browser to the hosted URL to visualize the Meshcat scene. Note that, unlike many visualizers, one cannot open the visualizer until this server is running.

Note: This code is currently a skeleton implementation; it only allows you to set (boolean) properties as a minimal demonstration of sending data from C++ to the viewer. It is the result of the second PR in a train of PRs that will establish the full Meshcat functionality. See #13038.
