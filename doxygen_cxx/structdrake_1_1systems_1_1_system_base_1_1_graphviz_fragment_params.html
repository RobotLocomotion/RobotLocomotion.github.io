SystemBase::GraphvizFragmentParams Struct Reference

Detailed Description

The arguments to DoGetGraphvizFragment().

These parameters constitute a polite request; a user's System's implementation of DoGetGraphvizFragment() is not strictly required to honor any of these parameters, but generally should attempt to honor as many as possible.
