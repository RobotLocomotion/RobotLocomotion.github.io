PerspectiveQuadraticCost Class Reference
Formulating and Solving Optimization Problems » Costs and Constraints

Detailed Description

If z = Ax + b, implements a cost of the form: (z_1^2 + z_2^2 + ...

  • z_{n-1}^2) / z_0. Note that this cost is convex when we additonally constrain z_0 > 0. It is treated as a generic nonlinear objective by most solvers.
