ScopedName drake::multibody::parsing::ParseScopedName ( const std::string &  full_name)

Attempts to find a name using the following scoping rules:

  • The delimiter "::" may appear zero or more times.
  • If one more delimiters are present, the full name is split by the last delimiter. The provided model instance name must exist.
