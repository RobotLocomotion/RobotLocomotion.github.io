MpcVariable Class Reference

Detailed Description

This class implements primal-dual variables for model predictive control QPs.

See mpc_data.h for the mathematical description. Stores variables and defines methods implementing useful operations.

Primal-dual variables have 4 fields:

  • z: Decision variables (x0,u0,x1,u1, ... xN,uN)
  • l: Co-states/equality duals (l0, ... ,lN)
  • v: Inequality duals (v0, ..., vN)
  • y: Inequality margins (y0, ..., yN)

length(z) = nz = (nx*nu)*(N+1) length(l) = nl = nx*(N+1) length(v) = nv = nc*(N+1) length(y) = nv = nc*(N+1)
