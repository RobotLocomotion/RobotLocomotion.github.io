◆ cartesian_axis_masks

string_unordered_map<Vector6d> cartesian_axis_masks

Map from fully-scoped frame names to their 6D spatial velocity axis mask.

Each Vector6d is a binary mask [ωx, ωy, ωz, vx, vy, vz] indicating which Cartesian velocity components (angular and translational) are being tracked. All elements must be set to either 0 or 1, and at least one element must be 1.

◆ posture_gain

double posture_gain {1.0}

The proportional gain matrix K is a diagonal matrix with the diagonal set to this value (i.e., all joints use the same posture_gain).

Must be non-negative.
