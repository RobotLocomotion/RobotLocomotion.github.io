DenseVariable Class Reference

Detailed Description

Implements primal-dual variables for inequality constrained QPs, see dense_data.h for a mathematical description.

This class stores variables and defines methods implementing useful operations.

Primal-dual variables have 3 components:

  • z: Decision variables
  • v: Inequality duals
  • y: Inequality margins

where length(z) = nz length(v) = nv length(y) = nv
