    Drake v1.26.0

    Back to Release Notes

    Announcements

    • macOS wheels on PyPI now support both Python 3.11 and Python 3.12 (#20616).
    • This is the last release that will support Ubuntu 20.04 (“Focal”).
      • End of life timelines are documented as part of the Supported Configurations.

    Breaking changes since v1.25.0

    • The FindSeparationCertificateOptions.num_threads property is now deprecated in Python, and removed in C++. Use parallelism instead (#20776).

    • RenderEngine no longer inherits ShapeReifier (#20873).

      • This might be a breaking change for code that implemented its own custom RenderEngine and was using this as the ShapeReifier receiver. To fix the break, inherit from ShapeReifier directly yourself. See #20873 for suggestions.

    Refer to our Drake Stability Guidelines for our policy on API changes.
